# News Data Analysis & AI Summarization

This project demonstrates an end-to-end data pipeline that extracts real-time news data from external APIs, processes and analyzes the data using Python, and applies AI-based summarization to generate insights from unstructured text.

The solution simulates a real-world data workflow, including data ingestion, cleaning, analysis, and visualization, showcasing practical skills relevant to data analyst and data science roles.

## Objectives
- Collect real-time news data from an API
- Clean and preprocess unstructured text data
- Generate article summaries using AI
- Explore trends and patterns through data analysis
- Visualize insights from the dataset

## Tools & Technologies
- Python
- pandas
- matplotlib
- News API
- OpenAI API
- Jupyter Notebook

## Project Structure
- `main.py` – Main script that runs the full data pipeline
- `news_analysis.csv` – Processed dataset with summaries and topics
- `news_topics.png` – Visualization of topic distribution
- `README.md` – Project documentation

## Workflow
1. Fetch news data from the API
2. Clean and structure the dataset
3. Summarize articles using AI
4. Analyze trends and patterns
5. Visualize findings

API keys are not included in this repository. Use environment variables or a `.env` file for credentials.
